# Useful OpenClaw (Clawdbot) Skills & Plugins

**Saved:** 2026-02-04

## Skills List

### 1. QMD Skill - 95% Token Reduction
- **URL:** https://github.com/levineam/qmd-skill
- **Purpose:** Cuts token usage by 95%
- **Impact:** Massive cost savings for free/paid tiers

### 2. Supermemory - Unlimited Memory
- **URL:** https://github.com/supermemoryai/clawdbot-supermemory
- **Purpose:** Give OpenClaw unlimited memory
- **Impact:** Better context, personalization

### 3. Prompt Guard - Injection Defense
- **URL:** https://clawdhub.com/seojoonkim/prompt-guard
- **Purpose:** Advanced prompt injection defense
- **Impact:** Security for multi-tenant platform (critical for Zaki)

### 4. Find Skills - Skill Discovery
- **URL:** https://clawdhub.com/JimLiuxinghai/find-skills
- **Purpose:** Helps users discover and install skills by asking "how do I do X"
- **Impact:** User self-service, reduces support

### 5. Don't Hack Me - Security Self-Check
- **URL:** https://clawdhub.com/peterokase42/dont-hack-me
- **Purpose:** Security self-check
- **Impact:** Platform hardening
