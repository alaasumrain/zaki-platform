## üöÄ How to Install

### Option 1: ClawHub CLI
```bash
# Install ClawHub CLI
npm install -g clawhub

# Install skills
clawhub sync git
clawhub sync github
clawhub sync self-improving-agent
clawhub sync openai-whisper-api
clawhub sync gitai-skill
clawhub sync nextjs-expert
clawhub sync typescript-lsp
```

### Option 2: Manual
1. Browse: https://clawhub.ai
2. Search for skill name
3. Download SKILL.md and files
4. Place in: `/root/clawd-user-{telegram-id}/skills/{skill-name}/`
